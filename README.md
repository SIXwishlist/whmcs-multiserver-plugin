# WHMCS Plesk Multi Server Provisioning Module #

## Summary ##

This provisioning module allows you to create (maintain) Plesk's Panel Multi Server accounts.

The list of the supported functions:

 * AdminLink
 * ClientArea
 * CreateAccount
 * SuspendAccount
 * UnsuspendAccount
 * TerminateAccount
 * ChangePassword
 * AdminServicesTabFields
 * ChangePackage
 * UsageUpdate
